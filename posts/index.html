博客更新教程

新建博客markdown文件，并编辑博客内容（文件名为**.md） hugo new "posts/new.md" 生成public目录 hugo -D 进入到public目录 cd public 添加当前目录下所有文件进暂存区 git add . 提交，即保存版本，加入-m参数不用进入填写更改信息界面 git commit -m "new blog added" 将本地仓库推送远程仓库分支master git push 焦鹏飞

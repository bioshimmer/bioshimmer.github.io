git知识点总结

February 17, 2023

Introduction

总结一下git中的常用操作：查看分支、删除分支、推送的操作

查看分支

git remote -v#查看远程仓库的信息
git branch aaa#创建aaa的分支
git branch #查看本地分支
git branch -a#查看本地和远程仓库分支

[!TIP]

当使用git branch -a无法显示远程分支时，尝试git fetch，该命令将获取远程服务器的索引。

远程推送

git push your_resositories your_branch

[!TIP] 远程库的名称是remote时指定的名称，默认为origin。

删除分支

git push your_resositories -d your_branch
